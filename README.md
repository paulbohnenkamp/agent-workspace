# Agent Platform

Agent Platform is a package-first, project-centric runtime for long-running human-agent collaboration.

![Hiring Workspace example](docs/images/projects/hiring-project.png)

It gives us:
- projects that organize context
- agents, tools, skills, channels, schedules, resources, and artifacts as filesystem packages
- an event-driven runtime with durable state, resumable work, and auditability
