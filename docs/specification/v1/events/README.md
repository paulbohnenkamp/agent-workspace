# Event Taxonomy

Event names SHOULD follow:

```text
<object>.<verb>
```

Examples:

```text
workspace.created
workspace.archived
artifact.created
artifact.updated
artifact.published
run.started
run.completed
run.failed
action.requested
action.completed
knowledge_source.added
participant.added
agent.started
```

## Event Categories

- Workspace Events
- Artifact Events
- Run Events
- Action Events
- Knowledge Events
- Participant Events
- Agent Events
- System Events
