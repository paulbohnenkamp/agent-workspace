# Object Lifecycles

## Workspace

```text
Defined → Instantiated → Active → Suspended → Archived
```

## Artifact

```text
Draft → Generating → Review → Published → Archived
```

## Run

```text
Pending → Executing → Waiting → Completed
                      ↘ Failed
```

## Action

```text
Available → Requested → Authorized → Executed → Completed
                              ↘ Rejected
```

## AgentSession

```text
Created → Running → Idle → Terminated
```
