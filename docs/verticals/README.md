# Vertical Workspaces

The vertical workspace examples validate that multiple business experiences can be represented through the same platform nouns.

## Included Verticals

- Decision Workspace
- Finance Workspace
- HR Workspace
- Partner Workspace

## Validation Claim

These are not four separate applications.

They are four workspace definitions rendered by one platform:

```text
WorkspaceDefinition
        ↓
Workspace Interpreter
        ↓
Workspace Shell
        ↓
Artifacts + Actions + Runs + Knowledge
```
